# adapter适配器

Python开发期间，在使用第三方数据库等场景时，常会遇到`apater`这个概念。

`driver驱动器` ≈ `adapter适配器` ≈ `binding粘合剂` ≈ `wrapper封装`

adapter/driver/binding/wrapper，指的是：

把一个别的功能，封装后，提供操作该功能的接口

起到了，作为粘合剂，把其他的库和你写的代码粘合起来效果
